# Panel del Agente de IA para Soporte

El **Panel del Agente de Inteligencia Artificial para Soporte al Cliente** es una aplicación web diseñada para que pueda gestionar, monitorear y optimizar la atención a sus clientes mediante agentes de voz y texto impulsados por IA.

Desde esta plataforma podrás:
- Configurar sus agentes inteligentes.
- Supervisar llamadas y conversaciones.
- Analizar estadísticas de uso.
- Gestionar mensajes de WhatsApp en un solo lugar.

---

## Funcionalidades principales

- **Estadísticas de llamadas**  
  Visualice métricas clave para evaluar el rendimiento de tu atención al cliente.

- **Registro de llamadas**  
  Acceda al historial de llamadas con sus respectivas transcripciones.

- **Configuración de agentes (voz y texto)**  
  Personalice el comportamiento de sus agentes de IA.

- **Administrador de WhatsApp**  
  Gestione conversaciones con sus clientes desde una interfaz centralizada.


![Panel de estadisticas de llamadas](/img/ai-agent-support-dashboard/ai-agent-stats-panel.png)

## Cómo iniciar

Para acceder a la plataforma es necesario completar un proceso de registro previo. Puede solicitarlo contactando a nuestro [equipo](https://edenia.com/contact-blockchain-developers).

Una vez que tenga sus credenciales:

1. Ingrese al sistema: https://ai-agent.edenia.cloud

2. Instale la aplicación en su dispositivo.

3. Inicie sesión con las credenciales proporcionadas.

4. Acepta las notificaciones cuando el sistema lo solicite.

5. ¡Listo! Ya puedes comenzar a gestionar sus agentes.

![Pantalla de login](/img/ai-agent-support-dashboard/login.png)

## Instalación de la aplicación

La plataforma funciona como una **PWA (Aplicación Web Progresiva)**, lo que permite instalarla fácilmente en su dispositivo.

### En dispositivos móviles (Android / iOS)

1. Abra el enlace en su navegador: https://ai-agent.edenia.cloud

2. Si aparece el mensaje de instalación, presiona **"Instalar"**.

3. Si no aparece:
   - Abra el menú del navegador (⋮)
   - Seleccione:
     - **"Agregar a pantalla de inicio"** o  
     - **"Instalar aplicación"**

## Notificaciones

Para recibir notificaciones en múltiples dispositivos:

- Debe instalar la aplicación en cada dispositivo.
- Iniciar sesión en cada uno.
- Aceptar los permisos de notificación cuando el sistema lo solicite.

Esto le permitirá recibir alertas en tiempo real cada vez que:
- Ingrese un nuevo mensaje.
- Se genere una nueva interacción entre el agente de IA y algun cliente.

![alt text](/img/ai-agent-support-dashboard/notifications.png)

### Si no recibe notificaciones

Algunos navegadores con configuraciones avanzadas de privacidad pueden bloquear las notificaciones automáticamente.

Si no recibe alertas:

- Verifique que las notificaciones estén habilitadas para este sitio.
- Revise que el navegador permita notificaciones en segundo plano.
- Intente reinstalar la aplicación y aceptar nuevamente los permisos.
- En algunos casos puede ser necesario habilitar el uso de servicios de notificaciones del navegador.

## Administrador de WhatsApp

El **Administrador de WhatsApp** es el espacio donde puede gestionar todas las conversaciones con sus clientes.

### ¿Qué puede hacer aquí?

- Ver todos los chats activos.
- Leer mensajes entrantes de clientes.
- Visualizar respuestas generadas por IA.
- Intervenir manualmente en la conversación.

Los agentes humanos pueden tomar el control de cualquier conversación en cualquier momento, asegurando una atención personalizada cuando sea necesario.

Además:
- Cada nuevo mensaje genera una notificación automática.
- Todos los administradores conectados recibirán alertas en sus dispositivos.

![Administrador de WhatsApp](/img/ai-agent-support-dashboard/whatsapp-manager.png)
